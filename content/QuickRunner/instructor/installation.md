QuickRunner目前有Linux，macOS，Windows三种不同的版本，同时支持X86，ARM两种不同的架构。
您可根据自身需要在 [GitHub Releases][releases] 页面中选择需要的版本,当前 QuickRunner 支持的环境包括：

- macOS(darwin) + amd64(x86)
- macOS(darwin) + arm64(M1)
- linux + amd64(x86)
- linux + arm64
- windows + amd64(x86)
